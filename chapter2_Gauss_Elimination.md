# singular case
singular case 란 되돌릴 수 없는 행렬을 말한다.<br>
예를들어 Ax = b 의 식에서 b 가 0이되는 경우 b 를 다시 A 로 돌릴 수 없다 따라서 해당 식은 singular case 이다<br>
또한 singular case 는 해가 없거나 무한대로 많은 특징을 가지고 있다.<br>
